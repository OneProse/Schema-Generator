## Features

- **7 Schema Types** — LocalBusiness, Organization, Service, FAQPage, BreadcrumbList, Product, Article
- **Zero Dependencies** — Pure HTML, CSS, and JavaScript in a single file
- **Works Offline** — Load once, use anywhere
- **Syntax-Highlighted Output** — Color-coded JSON-LD preview
- **Script Tag Toggle** — Switch between raw JSON and `<script>` wrapped output
- **Copy to Clipboard** — One-click copy with visual feedback
- **Validation** — Required field enforcement with inline errors
- **Dynamic Builders** — Add/remove rows for FAQs, breadcrumbs, and more
- **Responsive** — Two-column desktop layout, stacked on mobile
- **Dark Mode** — Professional, tool-like aesthetic

## Supported Schemas

| Schema | Use Case |
|--------|----------|
| **LocalBusiness** | Physical businesses with address, hours, and contact info |
| **Organization** | Companies, nonprofits, and brands with social profiles |
| **Service** | Professional services with pricing and service area |
| **FAQPage** | FAQ sections that qualify for rich results in Google |
| **BreadcrumbList** | Site navigation paths for enhanced search appearance |
| **Product** | E-commerce products with pricing, availability, and reviews |
| **Article** | Blog posts, news articles, and technical content |

## Getting Started

```bash
git clone https://github.com/your-username/schema-generator.git
cd schema-generator
```

Open `index.html` in your browser. That's it.

## Deploy to GitHub Pages

1. Push the repository to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your tool will be live at `https://your-username.github.io/schema-generator/`
